#### Specs
  - This application has the following features:
    - Hunger meter, happy meter, and discipline meter.
    - feed the pet will increase hunger meter.
    - happy meter will increase by playing with it and feeding it snacks.
    - discipline meter will increase when you use the "scold" button when it does not want to be fed or played with.
    - play : it'll increases the happy meter
    - sleep : it'll sleep , if you play with it long enough and happy meter is full.
    - die: no food, attention , not enough rest
    - Stages of life: Baby, Child, Teen, Adult, and Senior.
